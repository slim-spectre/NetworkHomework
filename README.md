# Звіт про виконання завдань з комп'ютерних мереж

## Завдання 1: Аналіз мережевої конфігурації системи

### 1. Мережеві інтерфейси та фізичні адреси
У системі виявлено **5 мережевих інтерфейсів** (включаючи фізичні та віртуальні адаптери).
**MAC-адреси:**
- `10-63-C8-55-27-AD`
- `12-63-C8-55-27-AD`
- `22-63-C8-55-27-AD`
- `98-FA-9B-C6-A7-4E`
- `10-63-C8-55-27-AE`

### 2. IP-адреса та підмережа
- **IPv4-адреса:** `192.168.0.102`
- **Маска підмережі:** `255.255.255.0`
- **Адреса мережі:** `192.168.0.0/24`

### 3. Шлюз за замовчуванням (Default Gateway)
- **Адреса шлюзу:** `192.168.0.1` (Адреса роутера).

### 4. ARP-кеш
В ARP-кеші зафіксовано **9 записів**:
- **1 запис:** Співвідношення IP/MAC для основного роутера.
- **8 записів:** Службові адреси (Multicast/Broadcast), що використовуються ОС Windows для виявлення пристроїв та мережевих сервісів.

## Завдання 2: Розрахунок параметрів підмереж

### 1. Підмережа `192.168.1.100/24`
- **Клас мережі:** C
- **Мережева адреса:** `192.168.1.0`
- **Broadcast адреса:** `192.168.1.255`
- **Кількість можливих хостів:** 254
- **Десяткова маска:** `255.255.255.0`

### 2. Підмережа `10.0.5.50/16`
- **Клас мережі:** A (з використанням маски класу B)
- **Мережева адреса:** `10.0.0.0`
- **Broadcast адреса:** `10.0.255.255`
- **Кількість можливих хостів:** 65 534
- **Десяткова маска:** `255.255.0.0`

### 3. Підмережа `172.16.8.200/20`
- **Клас мережі:** B (з використанням розширеної маски)
- **Мережева адреса:** `172.16.0.0`
- **Broadcast адреса:** `172.16.15.255`
- **Кількість можливих хостів:** 4 094
- **Десяткова маска:** `255.255.240.0`
